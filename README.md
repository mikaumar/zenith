# Zenith

**Public speaking confidence trainer**

Build speaking confidence through structured practice — flashcards, quizzes, real-world scenarios, and daily challenges.

## Features

- **Flashcards** — core speaking techniques and principles with spaced review
- **Quiz** — test your knowledge of delivery, structure, and persuasion
- **Scenarios** — practice with realistic speaking situations
- **Daily Challenge** — one new challenge every day to build consistency
- **Progress tracking** — XP, streaks, and performance history

## Tech

- React + TypeScript + Vite
- Tailwind CSS
- Framer Motion
- Canvas Confetti
